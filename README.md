# Late night shopping

Submission for the [Jame Gam Christmas Edition 2022](https://itch.io/jam/jame-gam-christmas-edition-2022).

You play as Santa, who needs to quickly go buy some presents because the elves were getting lazy that year.

The game is somewhat **incomplete**, since I didn't end up having enough time to draw the background and refine the menus.
